# 今天吃什么：家庭点餐小程序

这是一个微信原生小程序项目，使用微信云开发。当前处于阶段 0：项目初始化和基础骨架。

## 当前已经完成

- 小程序项目配置
- 云开发环境初始化配置
- 首页、家庭、点菜、购物车、共享餐单、采购单、我的页面骨架
- 默认餐次判断工具
- `login` 云函数骨架

## 如何在微信开发者工具里打开

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择本文件夹：
   `C:\Users\Administrator\Desktop\家庭点餐小程序`
4. AppID 使用项目里已经配置好的 AppID。
5. 导入后，确认顶部可以看到“云开发”入口。

## 第一次运行前需要做

### 1. 创建云数据库集合

打开微信开发者工具里的“云开发”，进入数据库，先创建集合：

```text
users
```

后续阶段还会继续创建：

```text
families
family_members
chef_recipes
family_dishes
meal_orders
purchase_lists
reviews
```

当前阶段只需要 `users`。

### 2. 上传 login 云函数

在微信开发者工具左侧找到：

```text
cloudfunctions/login
```

右键 `login` 文件夹，选择：

```text
上传并部署：云端安装依赖
```

完成后再编译小程序。

## 当前注意事项

- 首页会调用 `login` 云函数来验证微信云开发身份。
- 如果还没有创建 `users` 集合，或还没有上传 `login` 云函数，首页会提示登录验证失败。
- 这是正常的，按上面的步骤处理即可。
- `小程序开发资料.txt` 不会提交到 GitHub，里面可以保留你本地的配置记录。
